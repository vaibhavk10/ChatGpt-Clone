# ChatGPT Clone

A modern, responsive ChatGPT clone built with HTML, CSS, and JavaScript, powered by Google's Gemini API. This project features a clean interface similar to ChatGPT with chat history management and responsive design.

## Features

### Core Functionality
- Real-time chat interface with AI responses
- Chat history management
- New chat creation
- Individual chat deletion
- Full history clearing
- Persistent storage using localStorage

### User Interface
- Clean, modern design
- Dark theme
- Responsive layout
- Mobile-friendly design
- Smooth animations
- Loading indicators
- Custom scrollbar

### Technical Features
- Code block support
- Error handling
- Context maintenance
- Message formatting
- Local storage integration

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google's Gemini API
- Font Awesome Icons

## Setup and Usage

1. Clone the repository:
```bash
git clone https://github.com/vaibhavk10/ChatGpt-Clone.git
```

2. Open `index.html` in your browser

3. Start chatting!
